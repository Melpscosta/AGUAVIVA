# ÁguaViva

Landing page do ÁguaViva — uma onda de mudança na sustentabilidade marinha.

## Desenvolvimento

```bash
npm install
npm run dev
```

Abra [http://localhost:5173](http://localhost:5173) no navegador.

## Build

```bash
npm run build
npm run preview
```

## Estrutura

- **Hero** — título e tagline com navbar em glassmorphism
- **Game Placeholder** — espaço reservado para o jogo interativo
- **Scroll Indicator** — ícone de mouse com "SCROLL DOWN"
- **Como Funciona** — instruções com peixe-guia animado e botão CTA
- **Parcerias** — explicação, logos de parceiros e botão WhatsApp

## Configuração

Edite `src/config.ts` para alterar o número do WhatsApp e a mensagem pré-preenchida.
